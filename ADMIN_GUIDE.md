# Руководство по админ-панели

Инструкция по использованию административной панели Career Platform. Mobile-first интерфейс.

## Доступ

Админ-панель доступна только пользователям с ролью **ADMIN**.

- **URL:** `http://localhost:8081/admin` (или ваш домен + `/admin`)
- **Переход:** из профиля — кнопка «Административная панель»

## Навигация

- **Хедер** — на каждом экране: кнопка «← Назад» и ссылка «Меню» для возврата на главную админки.
- **Главная** — список разделов. Выберите нужный раздел.

## Разделы

### 1. Приглашения (Invites)

Создание и управление invite-ссылками для регистрации новых пользователей.

#### Создание приглашения

1. **Приглашения** → **Создать приглашение**
2. Выберите **роль**: Специалист или Администратор
3. Укажите **максимум использований** (по умолчанию 1)
4. При необходимости — **дату истечения** (формат ГГГГ-ММ-ДД)
5. Нажмите **Создать приглашение**
6. Ссылка скопируется в буфер обмена. Отправьте её приглашаемому.

#### Список приглашений

- Код, роль, использовано/макс, дата истечения
- **Копировать ссылку** — копирует полную ссылку в буфер
- **Деактивировать** — отключает invite (новые регистрации по нему невозможны)

**Формат ссылки:** `{origin}/register?invite=КОД`  
На web при копировании используется текущий домен (`window.location.origin`), поэтому ссылка всегда ведёт на ваш задеплойный фронт. Для мобильной админки настройте `EXPO_PUBLIC_APP_URL` в `.env`.

---

### 2. Управление пользователями

- **Список** — все пользователи с поиском по email/имени и фильтром по роли (Специалист / Администратор)
- **Сменить роль** — назначить пользователю роль Специалист или Администратор (нельзя изменить роль самому себе и последнему админу)
- **Заблокировать / Разблокировать** — блокировка входа для пользователя
- **Удалить** — мягкое удаление (деактивация) пользователя

---

### 3. Управление вакансиями

- **Список** — все вакансии с фильтрами
- **Создать** — новая вакансия
- **Детали** — просмотр, редактирование, деактивация

---

### 4. Управление рекомендациями

Пользователям показываются рекомендации по направлению и уровню профиля.

- **Список** — все записи с фильтрами
- **Создать** — direction, level, действия (шаги)
- **Детали** — просмотр, редактирование, удаление

---

### 5. Аналитика

- **Ключевые показатели**: пользователи, создано приглашений, активных приглашений, активировано приглашений, профили, просмотры вакансий
- **Воронка приглашений**: создано → открыто → регистрации → профили

Обновите данные — pull-to-refresh (потяните вниз).

---

## Роли при регистрации по invite

| Роль | Описание |
|------|----------|
| **Специалист** | Обычный пользователь (по умолчанию) |
| **Администратор** | Доступ к админ-панели |

## Советы (mobile-first)

- Используйте **«Меню»** в хедере для быстрого перехода между разделами
- **«← Назад»** возвращает на предыдущий экран
- На мобильном удобнее работать в портретной ориентации
